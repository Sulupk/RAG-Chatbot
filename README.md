# 📄 Document RAG Chatbot

An AI-powered Document RAG (Retrieval-Augmented Generation) chatbot that enables intelligent question answering from PDF documents. It uses FAISS for fast semantic search, HuggingFace embeddings for text understanding, and Groq-powered LLaMA models for generating accurate, context-aware responses, all integrated into a Streamlit interface.

---

## 🚀 Features
- Upload PDF documents
- Ask questions based on document content
- Context-aware and accurate answers
- Fast retrieval using FAISS
- Interactive UI using Streamlit

---

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace
- Groq (LLaMA)

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/your-username/RAG-Chatbot.git

2. Navigate to the project folder:
cd RAG-Chatbot

3. Install dependencies:
pip install -r requirements.txt

4. Add your API key

5. Run the application:
streamlit run app.py
